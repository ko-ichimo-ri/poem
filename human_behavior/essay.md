## 提案手法
今回思いついたアイデアを説明する。

人間中心主義的な考え方を用いず、人間の行動を10個に分類する。この分類を用いてそれぞれの行動に関する妥当性(ある種の正しさ)を考察する。同時に、従来人間特有の行動と考えられていた人文学・社会科学的な知見を一般化し、生成AIやロボット、家電、車、人間以外の生物など様々な主体を含めた抽象的な体系を目指す。

### 人間の捨象
コンピュータや家電、動植物、法人など人間以外も社会に含め、人間を特別視しない考えをする。人間以外のものも人間がしたいことを引き受けたり利害関係がある場合は社会の一員と考える。いや、この話に人間は登場しなくていい。社会性昆虫やロボットの一家は人間のいない社会である。いずれも入出力と処理があり、お互いに影響しあうという特徴がある。このようなものを「主体」と呼んでいる。本エッセイにおいて「主体」は「入力」を受け取り「出力」する「処理」を行う実体のことである。実体は一つのことも複数のこともあり、例えば、一人の人間や企業、家族、法人などは全て一つの「主体」とみなせる。

現在AIやロボットの台頭で人間と同様の知能を持つ別存在が現れようとしている。その中で人間中心的な考えであれば、AIやロボットの個性を尊重せず、擬人化してしまう恐れがある。例えば人間は心臓がドキドキしたり、喉が詰まるような感覚になったり、頬の筋肉が弛緩したり収縮したりする感覚を通じて感情を認識する。それに対しAIやロボットだと、仮想的な文字列であったりモーターの回転数だったりを感知して感情が芽生えるかもしれない。しかし、人間中心主義的であればその差異まで一般化してものを語れない可能性があるのだ。

このとき、最新技術ではなく素朴に現在の現実に沿わせて考える方がわかりやすいかもしれない。例えば、家電はそれまで人間がやっていた家事を人間の代わりに行ってくれている。洗濯機は持ち主がボタンで入力した設定や洗濯槽に入れた衣服や洗剤、柔軟剤などを「入力」として受け取り、モーターを制御して「処理」し、洗濯と脱水を終えた衣服を「出力」するのである。ここでは、人間が洗濯という家事を洗濯機に代替させているので人間と洗濯機の社会ができている。他社に仕事を外注するのに似たことを機械に対して行っているだけである。ペットや野生動物との関わりでは仕事を代替させるわけではないが、食べたり食べられたりの利害関係があるので社会といえよう。この時、相手の動物に対する情報を視覚や嗅覚などを用いて「入力」し、脳で「処理」したあと、威嚇したり捕食したりする行動を「出力」する「主体」である。

これまでの話に加えて、人間的な要素を捨象し、抽象的に俯瞰することで学問に新たな視点を提供できる。例えば、本エッセイでは、倫理は法ではなく思想の技術であるとか、社会科学は科学と技術の間であるとか、学歴を得るために勉強するのは学問ではなく経済活動だとか、盗難や殺人は経済活動とも捉えられるとかそういった主張を行う。


### 世界観とモデル

世界観は3段階で段階的に紹介し、概要を理解していただいたところで具体的なモデルを図を使って説明しよう。なお、モデルはカジュアルに紹介し、後の章[人間行動の形式化](#人間行動の形式化)にて厳密に形式化する方法を議論する。

#### 世界観1　～主体・処理・入出力～

人間が外部環境の中で自ら行動していく様を抽象化し、人間を捨象する。本節ではそんな世界観を具体的に説明する。

「主体」は人間の一般化である。外部から入る情報である「入出力」と自らの考えや行動を表す「処理」をつなぎ合わせる役割を「主体」は果たす。

「入出力」と「処理」を抽象化してみよう。ここでいう「入出力」は予め与えれている組のことで「処理」は入力に対し出力を返す一連の流れである(ソフト開発でいうインタフェースと実装)。「予め与えれている」と述べたが、運命論的な書き方であり、未来に全人類が予想していない事象が起こることも含まれる。「入出力」と「処理」に差異があるとき、「入出力」と「処理」のどちらを修正するかも人間行動の分類の上で重要になる。

人間の行動を思い返せば、五感で感じたことや脳内で考えたことをきっかけとしてなんらかの行動をおこなっている。その行動の結果は内部や外部となんらかの相互作用がある。その相互作用を起こすとき、妥当性がほしくなるときがある。例えば、(具体例準備中:例はいくつかあげ、分類が相異なるものにする)。このような妥当性は様々な基準があるが、行動の結果と相互作用の対象の差を埋めようとする共通点があるのではないだろうか。その妥当性に基づいて、差を埋めるとき何かしらの成長や発展があることも共通しているのではないだろうか。この妥当性と発展性に注目して人間活動を10個に分類するのが本エッセイの提案手法である。なお妥当性が不要なものは本エッセイでは取り扱わないが、本エッセイで定義づける妥当性は意味が広いのでたいていの活動に当てはまると考えられる。例えば、道で転んでしまったとか貧乏ゆすりをしてしまったなど偶発的・無意識的な行動は対象とならない。だが、転んだ後立ち上がるだとか、貧乏ゆすりをやめるように努力するだとかは本エッセイの対象である。他にも面白くてのめりこんでしまった趣味・部屋の片付けなどは本エッセイの対象となる。

「主体」は「入力」に対して「処理」を行い、「出力」する。この出力に相当する部分は「主体」の外部でも準備されている。外部で「入出力」のセットとして与えられた出力を「外部出力」と呼ぶ。「主体」は処理によって得られた「内部出力」を「外部出力」と比較し、差を小さくしようとする制御を行っている。仮説を立て(「処理」)、実験した後予想に反した結果が出た(「外部入出力」)ら、仮説を立て直すはずだ。この制御はフィードバック制御に似ているがもう少し広い概念である。フィードバック制御は「処理」を変更するのに対し、「主体」はそれだけでなく「外部入出力」を変更することもある(ただし、「主体」が意図していたのか、意図していたとして意図通りだったのかは問わない)。例えば、研究(「処理」)において新しい発見があれば、学会の定説(「外部入出力」)が変わるであろう。「処理」の「内部出力」か「外部入出力」で与えられた「外部出力」のどちらを固定し、どちらを変更するかが分類の肝である。固定される役割を「妥当性」、変更される役割を「発展性」と呼ぶことにしよう。

| 用語 | 定義 |
|---|---|
| 主体 | 「入力」を受け取り「内部出力」する「処理」を行う実体。実体は一つのことも複数のこともある。 |
| 外部入出力(入出力) | 「主体」が処理を開始したり、処理結果を評価するために用いる存在。本文では「外部」を省略することが多い。 |
| 入力 | 「外部入出力」で用意された「主体」への入力 |
| 外部出力 | 「外部」で用意された出力の期待値 |
| 内部出力 | 「処理」によって得られる出力 |
| 出力 | 「内部出力」または「外部出力」を指す用語。 |
| 処理 | 「入力」を「内部出力」に変換する動作 |
| 発展性 | 「内部出力」と「外部出力」を比較した後に変更される性質。「入出力」または「処理」の一方が持つ。 |
| 妥当性 | 「内部出力」と「外部出力」を比較した後に変更されない性質。「入出力」または「処理」の一方が持つ。 |

#### 世界観2　～個体・社会・客体～

ここでは「入出力」と「処理」の分類方法について概要を説明する。最初に分類の定義をカジュアルに説明し、その後その分類の意図と応用を説明する。カジュアルな定義の説明が長いが、そこを我慢すれば、「自由」とは何かに注目したそれぞれの長所・短所がわかるであろう。

「入出力」と「処理」は各々「主体」とどのように影響するかで分類する。「主体」による変更可能体およびその個数に注目して分類する。注目していただきたいのは定義の分け方はMECEになっていることである。

| 分類 | 定義 |
|--|--|
| 個体 | 「主体」が変更可能な一つのもの |
| 社会 | 「主体」が変更可能な二つ以上のもの |
| 客体 | 「主体」が変更不能なもの(変更可能な０個のもの) |

「個体」と「社会」における区別は主体が変更できるものが一つなのかそれより多いかと書いた。その区別は実は難しいが、「利害関係が生じる」「目的が複数ある」などで判定できる。

ここまでの話だけでは抽象的すぎるので豊富な具体例を見ていこう。

文脈により分類は変わりうる。例えば、企業は市場と比較すると「個体」であるが、社員と比較すると「社会」である。公理と推論規則から命題が証明できることも、証明自体の正当性を検証するうえでは「客体」といえるが、学生や数学者が試行錯誤しながら証明していく様は「個体」と言える。自然科学も再現性のある実験によって担保された理論は「客体」と言えるが、その再現性が認められるかどうかは「社会」と言える。数学も自然科学も研究として素晴らしいかどうかは「個体」および「社会」である。工学だって機械の動作は「客体」だが、便利さや安全性、環境への配慮などから作るものを決めたり基準を決めたりするのは「個体」および「社会」である。人間社会も普通の人からは「社会」であるが、世捨て人からは「客体」かもしれない。一人の人間だって個体に見えるが、過去の自分と今の自分で考えが異なる別人であって時間軸上の「社会」かもしれない。実際、過去の自分が書いた日記やソースコードのことを忘れ、今の自分が感動したり解読したりすることがある。これは親しすぎる家族・友人と対話しているようなものなので、「社会」とみなせるだろう。同様に歴史全般は時間軸上の「社会」と見做せそうだ。同じ時間の一人の心も素朴に考えれば「個体」であるが、異なる複数の考えが葛藤しているなら「社会」といえるかもしれない。つまりは、主体自身が「個体」であることも「社会」であることもある。友達同士で旅行を企画するとき、目的の相違や利害関係がないなら「個体」であるし、あるなら「社会」である。なんなら「個体」と「社会」を行ったり来たりすることもあるだろう。実際に仲間内の関係・状況がギスギスと仲良しで移り変わることもあるだろう。また、関係・状況が同じでも旅行に行くメンバーによって「個体」と感じたり「社会」と感じたりするかもしれない。さらに、同じ状況・同じ人であってもたまたま機嫌が良かったり悪かったりで分類が変わるかもしれない。旅行の話を少し抽象化すれば、主体が所属する組織は「個体」ともなりうるし、「社会」ともなりうるのである。そして、主体の違いによって、あるいは、同じ主体でも内部状態によって、同じ分類対象に対して「個体」「社会」「客体」のどれに分類するかは異なるのだ。このように本エッセイで取り扱うのは文脈依存な分類であり、これから読み進める上でそこに注意が必要である。例えば、下記の分類表で具体例があるが、文脈次第では分類が変わりうるということだ。

ただし、「入出力」や「処理」の**具体例が文脈依存**なだけであって、後で出てくる[人間行動の10分類](#人間行動の分類表)の**定義は文脈依存でない**ことに注意すること。論理学のアナロジーで説明すれば、定義は無定義述語の関係性で与えられる構文論・証明論的なものであり、具体例はその上のモデルという意味論的なものである。言語学の人にはシニフィアン(記号表現)とシニフィエ(記号内容)と言えば伝わるだろうか。いずれにせよ、解釈の対象だけで完結する不変なものか解釈次第で変わる可変なものなのかを意識して読み進めるべきなのだ。

さらにイメージを膨らませるために「入出力」「処理」の具体例を表に記す。いずれも文脈依存ではあるが、ニュアンスを伝えられるかと思うのでご覧いただきたい。見る上でのポイントは二つある。一つ目は変更可能なものの個数である。「個体」は一人や一つの団体で自由にできること、「社会」は利害関係者が複数いること、「客体」は人間にはどうにも変えられないことである。二つ目のポイントは「入出力」か「処理」かである。カジュアルにいえば、目指すべき目標が「入出力」、目標に向かう行動が「処理」と思っていただければよい。

| 入出力の分類 | 具体例 |
|---|---|
| 個体的入出力 | 空想した世界観、製品、企業理念・事業目標、個人の欲求・欲望、製品企画、道徳・倫理観、発明・発見 |
| 社会的入出力 | イデオロギー、法制度、他の動物(人間含む)の反応、社会通念、機械の操作と結果、娯楽、生活様式 |
| 客体的入出力 | 自然現象、実験・測定・調査の結果 |
| 個体的処理 | 空想、仕様作成・設計作成、業務、生活、表現技法・創作、思考 |
| 社会的処理 | 立法・司法・行政、貨幣・信頼による取引、暴力、権力・権威、選挙、文献解釈・論文執筆、SNS・メディア |
| 客体的処理 | 数学の証明、コンピュータによる計算、機械の動作 |

さて、分類の定義がわかったところで上記分類「個体」「社会」「客体」の意図とその応用についてお話ししよう。

突然であるが、学問の区別は何となくできるが明確に言葉にすることはできるだろうか。例えば、文系・理系という分け方について考えたことがあるだろうか。法学部は論理的なようで数学のそれとは違う。歴史学は事実を究明しているようで物理学のそれとは違う。同じ文系であっても、経済学部と法学部とで習うことは違うようだ。ここには一体どのような違いがあるのであろうか。

また、経済学において事実と価値の二分についてパトナムやデューイなどの哲学者が議論している。文系における「べき」の追究と理系における「である」の追究。その中間に位置する社会科学があるのだが、この中間とはなんであろうか。

さらに、「自由」という言葉の多義性である。国家や他者の介入を受けないことを「自由」ということもある。しかし、国家や他者がいなければ、却って不自由なこともある。例えば、コンビニでご飯を変えなければ自炊をしなければならない(その自炊の時間に大好きなアニメを見ることができたかもしれないのに・・・)。また、『入門・倫理学』[^3]によれば、哲学者カントは感情や欲望のまま行動するのは不自由であると考えていたそうだ。理性によって道徳的に行動し、人間が完全に自分を支配している状態をカントは「自由」と呼んでいる。これは自分以外の何者にも従わないという意味での自由だそうだ。「自由」とはいったい何であろうか？

[^3]: 『入門・倫理学』p40　赤林　朗・児玉　聡編(2018,勁草書房)

学問や我々の普段の行動を語るうえで、上記のように言葉にできない本質的な違いがあるであろう。実はこのような問題を解決するのが「個体」「社会」「客体」であると考えている。そして、その解決こそが私の意図であり、狙いである。実際にこの分類を応用して解決してみせよう。

次に「事実」や「価値」とは何かについて解決する。実はこの話は、「個体」「社会」「客体」のメリットを語るうえで欠かせない。だが、回りくどい方法を用いる。「もしそうでなければ、どのようなデメリットがあるか？」をここで語り、その後、「デメリットがないのがメリットです！」と続くのだ。(準備中)。余談だが、「人間」という言葉を持ち出していないに注目してほしい。「変更可能性」に注目した結果、人間性を捨象して「である」と「べき」を区別しているのである。

デメリットについて話した。最後は「自由」についてである。これは直接的に「個体」「社会」「客体」のメリットとなる。「デメリットがないのがメリットです！」という話である。(準備中。アイデアのみ箇条書きする。)。
- 判断:出力を比較した結果を二値で返すこと。妥当性が個体・社会・客体の場合、各々の返す値は好嫌・善悪・正誤である。

- 理性:事実や普遍性に向き合う主体の能力・性質

#### 世界観3　～主体と対象・通信ポート・存在～

(準備中)
「主体」は「入力の受け取り」→「サブ処理1」→「サブ処理2」→・・・→「サブ処理n」→「出力の受け渡し」という流れで「処理」を行う。この際、途中で処理を中断したり後回しにしたりすることもある。また、「主体」が複数の実体の集まりであるとき、一つの実体から他の実体に非同期に受け渡す必要もある。このとき、一時的に「処理」の途中の状態を記録するバッファのようなものを設ける。これを「非同期処理」という。「非同期処理」は以下のように分類できる。

#### モデル

(準備中。アイデアを箇条書きで書く。)
- 主体
  - 存在を送受信したり組み合わせたりできるもの。
- 対象とは何か
  - 主体を相手に存在をやりとり(少なくとも送信)するもの。
- 存在
  - 対象と主体でやりとりするもの。
- 人間行動中
  - 主体の振る舞いの一単位
- 入出力対象
  - 人間行動中に主体が変更しない対象。ただし、主体と独立に内部状態が変わることはある。
- 外部入出力
  - 人間行動中に入出力対象から得られる二つの存在の組。片方を外部入力、もう一つを外部出力という。外部を省略することも多々ある。
- 処理対象
  - 人間行動中に送受信可能な対象。
- 処理
  - 人間行動中に主体が外部入力を取得後、存在を処理対象と繰り返し送受信しあい、一つの存在を得ること。
- 過程(プロセス)
  - 処理を行う際に処理対象とやりとりした存在。
- 内部出力
  - 処理によって得た存在。文脈で明らかな場合、内部を略すこともある。
- 出力
  - 内部出力と外部出力の総称。なお文脈で明らかな場合は内部出力・外部出力の略を示すこともある。
- 出力元対象
  - 内部出力に対する処理対象または外部出力に対する入出力対象。
- 人間行動
  - 一つの人間行動中の入出力と処理の組であって、内部出力と外部出力を比較し、片方の出力に基づいてもう一方の出力元対象を変更すること。
- 発展性
  - 人間行動中に用いた処理対象および入出力対象に対して人間行動によって変更されるという性質。
- 妥当性
  - 人間行動中に用いた処理対象および入出力対象に対して人間行動によって変更されないという性質。
- 対象の内部状態
  - 対象が送信する存在が変わること
- 変更可能体とは何か
  - 対象が主体から存在を送受信する入り口のうち、その送受信によって対象の内部状態が変わりうるもの
- 変更不能体
  - 対象が主体と存在を送受信する入り口のうち、変更可能体でないもの
- 変更
  - 主体が変更可能体に存在を送信すること。
- 対象種: 変更可能体の数によって分類された対象の種類。
- 対象配置: 外部入出力対象か処理対象かで分類した対象の種類。

#### 形式モデルと意味論モデルへの架け橋
- 垂直結合:ある人間行動の発展対象を別の人間行動の妥当対象とすること(型があっていないといけない)。
- 水平結合:人間行動の出力(内部or外部)を別の人間行動の入力に渡すこと(型があっていないといけない)。以下例。
  - 物理の計算は心象による処理と論理による処理と法則による処理
- モデル:処理対象や入出力対象、存在の具体例
  - 第一意味論モデル: 形式モデルの「入出力対象」と「処理対象」に割り当てる二つの値(「個体」「社会」「客体」)と「発展性」「妥当性」を割り当てたもの。
  - 第二意味論モデル: 第一意味論モデルに具体例を当てはめることである。下記が例。
    - 科学の客体的入出力に哺乳類をとり、個体的処理を動物実験および観察と見做す。
  - 形式モデルとは別物。形式モデルは数理モデルの意味でモデル検査のモデル。意味論モデルは数理論理学の意味論の話。同じ用語だが別物。この混乱する用語は実際使われている(ソフトウェア工学の基礎より)。
- モデル採択: 具体例(モデル)を取ること。

各行動をさらに内部的にMECE的にわける方法を述べる。ここではホモ・サピエンスの視点の分類を行う。AIやロボット、宇宙人のような他の主体なら違う分類である可能性がある。

まずは内部分類に必要な用語を下記に記す。MECEになっていることに注意せよ。
  - 「個別論」は具体的で多種多様なものについてみていく。「一般論」は「個別論」の対象に共通する抽象的なただ一つのものについて探求する。
  - 「大規模」は人間の手で加工しにくいもの、「小規模」は人間の手で加工しやすいものを表す。人間がウルトラマンやスーパーマンなら、気象学が化学かもしれない。
  - 「生物」は人間に近いと感じる存在、「無生物」は人間から遠いと感じる存在。トイストーリーのウッディならおもちゃの学問かもしれない。
  - 「地球内」か「地球外」かは人間に距離的に近いかどうか。かぐや姫なら月を一つの学問にするかもしれない。
  - 「共時的」は同じ時間のこと、「通時的」は時間の流れがあること。つまり、時間が現在のみかそうでないかでMECEにわかれる。
  - 「言葉」は非同期処理として優秀な道具なので特別視。人間がそうしてきたので「人間への近さ」。「言葉」か「言葉以外」かを気にする。
  - 「既知」は人間が知っていること、「未知」は人間が知らないこと。これも「人間への近さ」
  - 「入出力」に注目するか「処理」に注目するか。これは今までの「人間行動の分類方法」の議論にでてきた二つの対象のうちの一つであり  MECE的である。
用語にある通り、内部分類は本エッセイで議論してきた「人間行動の分類方法」に加えて、「人間への近さや扱いやすさ」「一般論と個別論」「時間の流れ」で分類される。いずれにせよ、「[人間行動の分類表](#人間行動の分類表)」に記述された分類より細かくなれば、主体ごとに持つ個別的な性質に依存するようになる。そのため主体の一種である個性的なホモサピエンスであることを享受してこの節を読んでほしい。

まずは学問を内部分類しよう。
<pre>
学問
  |- 未知のもの
  |    `- 研究
  `- 既知のもの
       |- 教える側の視点
       |    `- 教育
       `- 教えられる側の視点
            `- 教養
</pre>
未知の知識や発見、発明などを探求するのが研究である。それに対し、既知の知識や発見、発明などを教えたり習得したりすることが教育・教養である。分け方は上記のものだけではなく具体的な入出力対象によりもっと分けることができる。それを「xについての学問」と呼ぶ。xを個体的入出力のモデルとするのである。

ここでさらに本質的にいえば「xについての学問」とは個体がxを行うあるいは行おうとしている[^1]人間である狭義の学問のことである。
[^1]: 歴史的アプローチや統計学的アプローチの自然科学は学問ではあるが科学ではないので「行おうとしている」を含めた。

以下に記していく。まずは大まかな分類からである。

  `- 社会科学: 数学・科学を除き、処理が社会であるものについての学問
</pre>

この分類により、新たな考察が捗るであろう。例えば、自然科学と社会科学の違いが明らかとなる。両者ともに、普遍性(変更の難しさ)の問題が付きまとうのだが、自然科学は我々人間から独立しているための普遍性であり、社会科学は関係者が多く合意を取るのが難しいためにおこる普遍性である。そのため、自然科学と違い社会科学は事実を見極めるだけでなく社会運動と結びつくし、社会科学の理論によって自己成就的予言や自己破壊的予言が起こるのである。また、社会科学と人文学の違いも明らかだ。人間行動において処理対象は営みのメインである。入出力を変えたり、入出力に合わせようとしたり、そのために労力がかかるのが処理なのだ。この処理が、個体だけの問題なのか、社会の問題なのかでアプローチが大きく変わる。個体であればその個性や個別の在り方が問われるのに対し、社会であれば他者と揉めないようにすることやトラブルを解決すること、他者へ要求を通すことに重点が置かれるのだ。人文学は前者、社会科学は後者に注目するので違いを感じるのは自然である。さらに、応用科学の人文学性にも気づけるであろう。高度な技術には自然科学や形式科学の知識が必要不可欠なため、応用科学はいわゆる理系というくくりになりがちである。しかし、答えているのは客体がどうあるかではなく、社会の要請に答えて自分たちがどうするかという問いなのである。これは自然科学や社会科学よりもまさに思想や芸術、文化の考え方に近いであろう。自然科学と形式科学の違いはもはや言うまでもない。客体が入出力対象か処理対象かという大きな違いがあるのだ。ここまで様々な例を挙げたが、MECE的に綺麗に分けることで我々が普段学んでいた考え方を深く理解できるようになる。

さて、上記の分類であるが、普段目にする分野の名前が出てこない場合がある。しかし、それは上記の分類に対して少しの補足で補うことができる。あるいは、分野ごとに専門家と学会の歴史依存となっている。このあと、実例を見ていこう。

同じものに対して別名を付ける場合がありうる。いくつか例を挙げる。
- 形式科学を「数学」と呼ぶこともある。
- 人文学を「人文科学」と呼ぶこともある。

教育機関の科目名と合わないところもあるが、上記の分類を組み合わせたり分けたりすれば対応できる。いくつか例を挙げる。
- 小学校
  - 初等的な数理科学とその周辺を合わせたものを「算数」と呼ぶ。単位を習って実際に測ることや文章題を式にすること、数列から規則を見出すことは、純粋数学ではありえず、理論物理や理論経済などの数理科学のセンスである。生活において数学的な概念は現実のものとの対応付けが大事であり、初等教育では純粋数学よりも数理科学のほうが良い題材であろう。
  - 初等的な学際分野を「総合」と呼ぶ。
- 中学・高校
  - 工学は「技術」と呼ぶ。
  - 人類学は「家庭」と呼ぶ。
  - 医療系は「保健体育」と呼ぶ。
  - 芸術学は聴覚の「音楽」と視覚の「美術」に分かれる。これは人間がメインで使っているセンサが聴覚・視覚だからであろう。
  - 人文学のうち日本語・英語に関するものを「国語」「英語」と呼んでいる。国内や世界における「人間への近さ」によるものであろう。
    - 「英語」ではなく「外国語」でほかの言語を履修できることもある。
  - 歴史学は「日本史」「世界史」に分かれる。国内や世界における「人間への近さ」によるものであろう。
  - 法学は「公民」と呼ぶ。
  - 哲学は「倫理」と呼ぶ。
  - 政治学と経済学を合わせて「政治経済」と呼ぶ。社会的処理が妥当性になっているところが類似している。
  - 地球科学と天文学を合わせて「地学」と呼ぶ。大規模な無生物という共通点がある。
- 大学
  - 自然科学と形式科学を合わせて「理学」と呼ぶ
  - 経営学と経済学をまとめて「経済学」と呼ぶこともある
  - 法学と政治学をまとめて「法学」と呼ぶこともある

さらに細かい内部分類を考えることができるが、それは各専門家に任せる。しかし、「各時代の人間への近さ」が積み重なった歴史で別れているのは共通しているであろう。逆に各々の時代の人間や歴史が異なれば異なる分類になっているのである。以下に例を挙げよう。
  - 数学は幾何、解析、代数など。物理は力学、電磁気学、熱力学など。分類は歴史を紐解く必要がある。
  - 芸術は「音楽」と「美術」がメインである。これは人間が聴覚と視覚に優れた生き物だからであろう。「人間への近さ」である。視覚や聴覚より触覚や嗅覚、味覚が勝る生物の文明であればそれらが「音楽」「美術」より台頭していたであろう（味覚も優れているため料理の発展も著しいが）。

さて、その他の人間行動の内部分類は上記で完結している。「xについての学問」の分類が「xそのもの」の分類と対応しているのだ。この学問の分類は対象となるxの自然な内部分類となる。なぜなら、過去の天才たちが大勢集まり長い歴史の中で洗練されたものの分類だからだ。例えば、自然哲学から洗練されて生まれた「自然科学」の分類によって科学が物理、生物、化学などに自然と分かれるのだ。そのため学問の対象と学問で分類が大きく異ならないと考え、学問の分類のみにとどめるとしよう。

### 対象と存在の正体

「入出力」と「処理」の関係を再考する。(この章は準備中である。後ほど詳しく書くが大きな間違いはない)。

| 対象の種類 | 対象名 | 存在名 | ルール | 文章 | 数値 | 理由 |
|---|---|---|---|---|---|---|
| 個体的入出力 | 認識 | 心象 | 心理 | 物語(覚書) | 目標値 | 動機 |
| 社会的入出力 | 合意 | 象徴 | 規範 | 文書(議論) | 基準値 | 要件 |
| 客体的入出力 | 現象 | 資料 | 法則 | 説明(記録) | 測定値 | 原因 |
| 個体的処理 | 表現 | 制作物 | 技法 | 文芸 | 達成値 | 仕組み |
| 社会的処理 | 説得 | 意思伝達媒体[^communication] | 弁論術 | 論述 | 得点 | 論拠 |
| 客体的処理 | 仮想 | 計算 | 論理 | 証明 | 理論値 | 前提 |

[^communication]: 意思伝達媒体はコミュニケーション・メディアである。他が漢字なのでそろえた。

他の概念にも適用が考えられる。例えば、ルールを得るための方法もある。科学なら帰納、数学なら演繹である。思想や芸術などにも相当するものがあるのだろう。これについては今後の課題とする。

| 存在の種類| 存在の源 | 定義 | 例 |
|---|---|---|---|
| 心象 | 認識 |  | 歴史、法解釈、数理モデル、物理の計算 |
| 象徴 | 合意 |  | 法の学説、法制度の制定、企画書 |
| 資料 | 現象 |  | 分類、史料、標本、写真、動画、論文、説明 |
| 制作物 | 表現 |  | 絵画、音楽、文芸、芸術としての写真・動画 |
| 意思伝達媒体[^communication] | 説得 |  | 歴史、法解釈、数理モデル、物理の計算、法の学説、法制度の制定、企画書 |
| 計算 | 仮想 |  | 演算、証明、シミュレーション、機械の動作 |

上記の分類を考えることで、微妙に異なる処理を明確に区別できる。いくつか日常的に言われている分類と比べて特徴的な例を詳しく見ていこう。物理の計算は「計算」に見えて「心象」である。なぜなら、数式に出てくる数字をどのように測定するかを念頭に置きながら近似をしたり仮定を追加する作業を平気で行っているからである。そのため純粋な「計算」を行う数学が得意でも物理が不得意であることがありうるし、その逆もしかりである。また、算数教育において演算方法だけでなく、おはじきやそろばんを使って自然数やその演算を説明したり、日常を舞台とした文章題を出したり、単位や四捨五入など測定に関する知識を教えたりしている。つまり、算数は「計算」だけではなく数字の「心象」も含んだ教育であり、ある意味数学と物理の合いの子なのである。

「掛け算の順序問題」がSNSで話題になる。たとえば、「ビスケットが3枚乗ったお皿が5皿あります。このときビスケット何枚ですか。」という問題で$` 3\times 5 `$のみが正解で$` 5\times 3 `$は誤答であるという主張があるのである。私はこの主張に反対だが、しかし、反論として自然数の加法の交換法則をナンセンスだと考えている。なぜなら、自然数の公理としてよく用いられるペアノ算術において加法の交換法則は非自明な定理だからである。ここでは「計算」ではなく「解釈」の問題ととらえ、各皿の1,2,3番目のビスケットに5皿対応させる解釈や$` (掛ける数)\times (掛けられる数) `$ではない数理モデル$` (掛けられる数)\times (掛ける数) `$の正当性に言及すべきであろう。また物理への橋掛けとして単位を重視し$` 3[ビスケット/皿]\times 5[皿] `$や$` 5[皿]\times 3[ビスケット/皿] `$などと書かせるとよいだろう。

このようにして混同しがちな処理を明確に区別し、議論が充実するのである。

この「非同期処理」を記号に置き換えたものが「言葉」である。記号とは対象を記録するために簡易的な別のものに置き換えたものである。記号は現物を残せないときや抽象的に考えたいときに便利である。抽象的に考えると、扱う対象の共通点や構造が見えることがあり、記号化する価値がある。「言葉」は以下のように分類できる。

| 言葉の分類 | 定義 | 例 |
|---|---|---|
| 物語(覚書) |  |  |
| 文書(議論) |  |  |
| 説明(記録) |  | 手続き、データ、記述、測定結果 |
| 文芸 |  | 擬人法、比喩・象徴、倒置法、冗談 |
| 論述 |  | 修辞的譲歩、例示、対比、強調 |
| 証明 |  | 否定、連言、選言、含意、順次・分岐・反復 |

どこを仲介する場所を考えれば、巷の混乱を整理し、明瞭な区別がつくようになるだろう。例えば、数学が得意だが、現代文の評論が苦手な人や逆に国語は得意なのに数学が苦手な人へ疑問を持たなくなる。ここには「証明」と「説明」、さらには「論述」に明確な違いがあるのだ。この例を掘り下げてみよう。

花子さんと太郎さんがそれぞれみかんを2個と3個持っている。合計は$` 2+3=5 `$でみかん5つである。他の問題を考える。花子さんと太郎さんがそれぞれりんごを2個と3個持っている。合計は$` 2+3=5 `$でりんご5つである。このように$` 2+3=5 `$を考える際、その式が表すものがみかんであろうがりんごであろうがどうでもよいのだ。だが、「みかんをください」と言われたときに「それはりんごが欲しいということですか？」と聞き返すのはナンセンスである。この違いはなんであろうか。$` 2+3=5 `$というのは数字の関係性にのみ注目しているが、「みかんをください」は現実に存在する黄色くて丸い果物を指している。つまり、記号と記号を結び付けたいのか、記号と現実を結び付けたいのかが異なるのだ。数学において、ときに現実を忘れ記号の操作のみに注目することで意外なもの同士が同じ構造を持つことに気づくことがある。これが「証明」の強力さだ。対して、「みかん」という単語を聞いて、その立体感や感触、匂いを思い浮かべながら説明する「説明」は日常生活で必須の技であることは明らかであろう。

このように言葉の分類に注目して物事を整理すれば、各々の性質を正確に捉えて、よりよい言葉の運用ができるのである。

言葉の性質として要素還元的に分解し、組み合わせられるという特徴がある。言葉を要約して理解しやすくするために、分解してできた言葉の集まりを「ルール」という。

| ルールの分類 | 定義 | 例 |

- 存在は対象種、対象配置で分類。

- ルール: 対象から得られる存在を要素還元主義的に分解して言葉にしたもの。

  - 要件: 規範が理由になったもの

**光速度不変の原理**も**地動説**も計算を楽にする規範であって、法則ではないのだ！
- 光速度不変の原理: 時計や定規を基準に測定するのではなく光速を基準にして測定しましょうという規範

  - 厳密にいえば、太陽と地球を一つに見た時の重心を中心にして太陽も地球も回ってるので地動説も近似である
  - 天動説だと円には近似できないのですごくナンセンスではある。
