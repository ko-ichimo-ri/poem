# 論旨
生成AIやロボットなどの台頭を意識して、人間という概念を用いず、入出力と処理の種類によってMECE的に人間の行動を10個に分類する。10分類により区別が難しい人間活動の違いが人間中心主義的な概念を用いずに明確になる。例えば、自然科学と社会科学、技術と科学、法と経済、法と政治、数学と科学などの区別が明瞭になる。明瞭化はそれぞれの人間活動について事例の妥当性を正確にとらえるのに役立つ。

人類行動の10分類の目標は変わりゆく文脈を正確に把握し、議論の混乱を防ぐ特効薬を生み出すことである。この特効薬は人間中心的な考えや歴史に囚われた慣習を抽象化・一般化する効果があり、人間と異なる知性への新たな眼差しを提供し、学問の新たな分野が自然と湧き出させる。

# 提案手法
今回思いついたアイデアを説明する。

人間中心主義的な考え方を用いず、人間の行動を10個に分類する。この分類を用いてそれぞれの行動に関する妥当性(ある種の正しさ)を考察する。同時に、従来人間特有の行動と考えられていた人文学・社会科学的な知見を一般化し、生成AIやロボット、家電、車、人間以外の生物など様々な主体を含めた抽象的な体系を目指す。

## 人間の捨象
コンピュータや家電、動植物、法人など人間以外も社会に含め、人間を特別視しない考えをする。人間以外のものも人間がしたいことを引き受けたり利害関係がある場合は社会の一員と考える。いや、この話に人間は登場しなくていい。社会性昆虫やロボットの一家は人間のいない社会である。いずれも入出力と処理があり、お互いに影響しあうという特徴があり、これを「主体」と呼ぶことにする。

「主体」を導入する必要性とその定義について述べていく。(準備中)

## 入出力と処理
「入出力」と「処理」を人間行動の本質とする。ここでいう「入出力」は予め与えれている組のことで「処理」は入力に対し出力を返す動作または計画である(ソフト開発でいうインタフェースと実装)。「予め与えれている」と述べたが、運命論的な書き方であり、未来に全人類が予想していない事象が起こることも含まれる。入出力と処理に差異があるとき、入出力と処理のどちらを修正するかも人間行動の分類の上で重要になる。

人間の行動を思い返せば、五感で感じたことや脳内で考えたことをきっかけとしてなんらかの行動をおこなっている。その行動の結果は内部や外部となんらかの相互作用がある。その相互作用を起こすとき、妥当性がほしくなるときがある。例えば、(具体例準備中:例はいくつかあげ、分類が相異なるものにする)。このような妥当性は様々な基準があるが、行動の結果と相互作用の対象の差を埋めようとする共通点があるのではないだろうか。その妥当性に基づいて、差を埋めるとき何かしらの成長や発展があることも共通しているのではないだろうか。この妥当性と発展性に注目して人間活動を10個に分類するのが本エッセイの提案手法である。なお妥当性が不要なものは本エッセイでは取り扱わないが、本エッセイで定義づける妥当性は意味が広いのでたいていの活動に当てはまると考えられる。例えば、道で転んでしまったとか貧乏ゆすりをしてしまったなど偶発的・無意識的な行動は対象とならない。だが、転んだ後立ち上がるだとか、貧乏ゆすりをやめるように努力するだとかは本エッセイの対象である。他にも面白くてのめりこんでしまった趣味・部屋の片付けなどは本エッセイの対象となる。

「主体」は「入力」に対して「処理」を行い、「出力」する。この出力に相当する部分は「主体」の外部でも準備されている。外部で「入出力」のセットとして与えられた出力を「目標値」と呼ぶ。「主体」は処理によって得られた「出力」を「目標値」と比較し、差を小さくしようとする制御を行っている。この制御はフィードバック制御に似ているがもう少し広い概念である。フィードバック制御は出力側を変更するのに対し、「主体」はそれだけでなく「目標値」を変更できる(ただし、「主体」が意図していたのか、意図していたとして意図通りだったのかは問わない)。「処理」の「出力」か「入出力」で与えられた「目標値」のどちらを固定し、どちらを変更するかが分類の肝である。固定される役割を「妥当性」、変更される役割を「発展性」と呼ぶことにしよう。

表. 用語の整理 (循環論法に見えるのは無定義述語だからである(要論証))
| 用語 | 定義 |
|---|---|
| 主体 | 「入力」を受け取り「出力」する「処理」を行う実体。実体は一つのことも複数のこともある。 |
| 外部 | 「主体」を一つ固定したときのそれ以外 |
| 入力 | 「外部」で用意された「主体」への入力 |
| 目標値 | 「外部」で用意された出力の期待値 |
| 主体の出力 | 「主体」の変換によって得られる出力 |
| 出力 | 「目標値」または「主体の出力」を指す用語。特に断りのない限り、「主体の出力」を指す。 |
| 処理 | 「入力」を「主体の出力」に変換する動作 |
| 入出力 | 「入力」と「目標値」のセット。「外部」で用意されているニュアンスがある。 |

入出力と処理は各々「主体」との関係で分類する。「主体」による変更可能性および個数に注目して分類する。

定義の分け方はMECEになっており、さらに下記の性質で特徴づけられる。
- 合意の必要性
- 主体による変更の難易度

上記性質は定義に基づいて実際の現象を観察すると納得できるものである(演繹的に導出できるかは不明)。この性質を知っておくと概念を混同した際の原因を特定しやすい。なぜなら片方の性質にのみ注目して混同していることが多いからである。

| 分類 | 定義 | 合意の必要性 | 変更の難易度 |
|--|--|--|--|
| 個体 | 「主体」が変更可能な一つのもの | 不要 | 易しい |
| 社会 | 「主体」が変更可能な二つ以上のもの | 必要 | 難しい |
| 客体 | 「主体」が変更不能なもの | 不要 | 難しい |

「個体」と「社会」における区別はものが一つか二つ以上かである。その区別は「利害関係が生じる」「目的が複数ある」などで判定できる。

文脈により分類は変わりうる。例えば、企業は市場と比較すると「個体」であるが、社員と比較すると「社会」である。公理と推論規則から命題が証明できることも、証明自体の正当性を検証するうえでは「客体」といえるが、学生や数学者が試行錯誤しながら証明していく様は「個体」と言える。自然科学も再現性のある実験によって担保された理論は「客体」と言えるが、その再現性が認められるかどうかは「社会」と言える。数学も自然科学も研究として素晴らしいかどうかは「個体」および「社会」である。工学だって機械の動作は「客体」だが、便利さや安全性、環境への配慮などから作るものを決めたり基準を決めたりするのは「個体」および「社会」である。人間社会も普通の人からは「社会」であるが、世捨て人からは「客体」かもしれない。一人の心も素朴に考えれば「個体」であるが、異なる複数の考えが葛藤しているなら「社会」といえるかもしれない。つまりは、主体自身が「個体」であることも「社会」であることもある。友達同士で旅行を企画するとき、目的の相違や利害関係がないなら「個体」であるし、あるなら「社会」である。なんなら「個体」と「社会」を行ったり来たりすることもあるだろう。実際に仲間内の関係・状況がギスギスと仲良しで移り変わることもあるだろう。また、関係・状況が同じでも旅行に行くメンバーによって「個体」と感じたり「社会」と感じたりするかもしれない。さらに、同じ状況・同じ人であってもたまたま機嫌が良かったり悪かったりで分類が変わるかもしれない。旅行の話を少し抽象化すれば、主体が所属する組織は「個体」ともなりうるし、「社会」ともなりうるのである。そして、主体の違いによって、あるいは、同じ主体でも内部状態によって、同じ分類対象に対して「個体」「社会」「客体」のどれに分類するかは異なるのだ。このように本エッセイで取り扱うのは文脈依存な分類であり、これから読み進める上でそこに注意が必要である。例えば、下記の分類表で具体例があるが、文脈次第では分類が変わりうるということだ。

ただし、「入出力」や「処理」の**具体例が文脈依存**なだけであって、後で出てくる[人間行動の10分類](#人間行動の分類表)の**定義は文脈依存でない**ことに注意すること。論理学のアナロジーで説明すれば、定義は無定義述語の関係性で与えられる構文論・証明論的なものであり、具体例はその上のモデルという意味論的なものである。言語学の人にはシニフィアン(記号表現)とシニフィエ(記号内容)と言えば伝わるだろうか。いずれにせよ、解釈の対象だけで完結する不変なものか解釈次第で変わる可変なものなのかを意識して読み進めるべきなのだ。

| 入出力の分類 | 具体例 |
|---|---|
| 個体的入出力 | 空想した世界観、製品、企業活動、個人の欲求・欲望、作品、道徳・倫理観、発明・発見 |
| 社会的入出力 | イデオロギー、法制度、他の動物(人間含む)の反応、社会通念、機械の操作と結果、娯楽、生活様式 |
| 客体的入出力 | 自然現象、実験・測定・調査の結果 |

| 処理の分類 | 具体例 |
|---|---|
| 個体的処理 | 空想、仕様・設計、業務、生活、表現技法・創作、思考 |
| 社会的処理 | 立法・司法・制度運用、貨幣・信頼による取引、暴力、権力・権威、選挙、文献解釈・論文執筆、SNS・メディア |
| 客体的処理 | 数学の証明、コンピュータによる計算、機械の動作 |

「人間」や「価値判断」という言葉を持ち出していないのが既存の考え方との違いである。「変更可能性」に注目した結果、人間性を捨象して「である」と「べき」を区別できた。

## 人間行動を非同期に行う方法

「入出力」と「処理」の関係を再考する。

「主体」は「入力の受け取り」→「サブ処理1」→「サブ処理2」→・・・→「サブ処理n」→「出力の受け渡し」という流れで「処理」を行う。この際、途中で処理を中断したり後回しにしたりすることもある。また、「主体」が複数の実体の集まりであるとき、一つの実体から他の実体に非同期に受け渡す必要もある。このとき、一時的に「処理」の途中の状態を記録するバッファのようなものを設ける。これを「非同期処理」という。「非同期処理」は以下のように分類できる。

| 非同期処理の分類 | 定義 | 例 |
|---|---|---|
| 計算 | 「サブ処理」と「サブ処理」の間のバッファまたはそのバッファを作ること | 演算、証明、シミュレーション、機械の動作 |
| 博物 | 「入力」と「処理」の間のバッファまたはそのバッファを作ること | 分類、史料、標本、写真、動画、論文、説明 |
| 表現 | 「処理」と「出力」の間のバッファまたはそのバッファを作ること | 絵画、音楽、文芸、芸術としての写真・動画 |
| 解釈 | 「博物」と「表現」の繰り返しによる疑似的な「計算」。複数の「処理」が連結して一つに見えるが、**「主体」だけでは完結せずに間に「外部」を挟む。** | 歴史、法解釈、数理モデル、物理の計算 |

(それぞれの詳しい話は準備中)

この「非同期処理」を記号に置き換えたものが「言葉」である。記号とは対象を記録するために簡易的な別のものに置き換えたものである。記号は現物を残せないときや抽象的に考えたいときに便利である。抽象的に考えると、扱う対象の共通点や構造が見えることがあり、記号化する価値がある。「言葉」は以下のように分類できる。

| 言葉の分類 | 定義 | 例 |
|---|---|---|
| 論理 | 処理と処理を仲介する言葉 | 否定、連言、選言、含意、順次・分岐・反復 |
| 説明的修辞 | 入力と処理を仲介する言葉 | 手続き、データ、記述、測定結果 |
| 文芸的修辞 | 処理と出力を仲介する言葉 | 擬人法、比喩・象徴、倒置法、冗談 |
| 評論的修辞 | 解釈に用いる言葉 | 修辞的譲歩、例示、対比、強調 |

どこを仲介する場所を考えれば、巷の混乱を整理し、明瞭な区別がつくようになるだろう。例えば、数学が得意だが、現代文の評論が苦手な人や逆に国語は特になのに数学が苦手な人へ疑問を持たなくなる。「論理」と「評論的修辞」は別物なのである。ロジカルシンキングと言いながら「論理」ではなく「説明的修辞」や「文芸的修辞」について述べている人が多い。批判的思考は言葉を使って頭を働かせているが、それは「論理」ではなく「説明的修辞」や「文芸的修辞」を繰り返し行う「解釈」の営みなのである。また、メタ的なことを言えばこのエッセイは説明的修辞と少しの文芸的修辞で成り立っているのだ(こういう余計なことをいうのは文芸的修辞である。いや、わかりやすい具体例と思えば説明的修辞か)。

言葉の性質として要素還元的に分解し、組み合わせられるという特徴がある。言葉を要約して理解しやすくするために、分解してできた言葉の集まりを「ルール」という。

| ルールの分類 | 定義 | 例 |
|---|---|---|---|
| 公理 | 処理と処理を繋ぐルール | ユークリッド幾何の公理、群・環・体の公理 |
| 法則 | 入力と処理を繋ぐルール | 運動方程式、作用反作用の法則、マクスウェル方程式 |
| 技法 | 処理と出力を繋ぐルール | 回想、起承転結、てんどん |
| 規範 | 解釈を要約したルール | 法律、制度、マナー、弁証法、規格、**光速度不変の原理**、**地動説・天動説** |

**光速度不変の原理**も**地動説**も計算を楽にする規範であって、法則ではないのだ！
- 光速度不変の原理: 時計や定規を基準に測定するのではなく光速を基準にして測定しましょうという規範
  - それにたぶんローレンツ流の考え方でも計算力がバチバチにあるなら現象を予言できる(ほんまか？)
- 地動説: 太陽が地球の周りを回ってるとすると計算が複雑すぎるので太陽を中心に地球が回っているように計算しましょうという規範。なお、回っているの定義を「円に近似できる」とするなら規範ではなく法則である。
  - 厳密にいえば、太陽と地球を一つに見た時の重心を中心にして太陽も地球も回ってるので地動説も近似である
  - 天動説だと円には近似できないのですごくナンセンスではある。
